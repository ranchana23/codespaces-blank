## Usage

### Run Locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open `http://localhost:3000`.

### GitHub Pages
This project is ready for GitHub Pages.
1. Push to GitHub.
2. Enable GitHub Pages for the **root** directory.
